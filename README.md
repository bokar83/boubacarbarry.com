## Run locally
Just open index.html in a browser or run python3 -m http.server 8000.

## Deploy (Hostinger)
1. Upload repo contents to Hostinger file manager (root folder).
2. Ensure index.html is root entrypoint.
3. Enable gzip/Brotli compression if available.
4. Update DNS for apex + www to Hostinger.
